# PaymentSplitter (PaymentSplitter.sol)

View Source: [contracts/payment/PaymentSplitter.sol](../contracts/payment/PaymentSplitter.sol)

**PaymentSplitter**

This contract can be used when payments need to be received by a group
of people and split proportionately to some number of shares they own.
